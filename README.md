# Knowledge-Card-Generator
The Knowledge Card Generator is an AI-based application that converts lengthy text, notes, or documents into short and easy-to-understand knowledge cards. It uses NLP techniques for summarization and keyword extraction, helping students in quick revision, learning, and exam preparation efficiently.
